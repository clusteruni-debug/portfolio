# Dossiers

This folder contains durable internal project dossiers for the portfolio.

Each dossier should follow `../schema.md` and stay evidence-backed. The content
may be used later to draft a public portfolio story, but the dossier itself is
not public copy.

The sections should be reconstructed from claim-level evidence. Raw Claude Code
JSONL, Codex raw logs, rollout summaries, and diary notes are not a separate
"utterance dump"; they are the basis for why-started, assumption, AI-interaction,
decision, status, and lesson claims.

Language rule:

- Korean is the primary review language.
- English is kept for machine-readable identity, evidence, freshness, and a short
  machine summary.
- Do not write only-English dossiers unless the user explicitly asks for it.
