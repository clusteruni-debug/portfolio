# Portfolio

Story-driven personal site. Built with Next.js 16 (App Router), TypeScript, Tailwind CSS v4, Framer Motion, Supabase, and TipTap.

## Setup

```bash
npm install
cp .env.local.example .env.local  # Add Supabase credentials
npm run dev                        # http://localhost:5110
```

## Deployment

Vercel (auto-deploy on push). GitHub repo: `clusteruni-debug/portfolio`
